CSS Style Sheets

Inline styles

Added to the style attribute of an HTML element.Inline styles take precedence over internal or external style sheets.

Internal styles

A set of rules added to the HEAD element of an HTML page inside STYLE tag.These styles are page-specific

External style sheet

A set of rules created as a separate .css file and applied using a style link attribute in the HTML page.

There are two ways to connect to an external style sheet: through the link HTML element or through the CSS @import command.
